# Projeto Final AOC

Dungeon Crawler feito em MARS

Para executar:
- Coloque o arquivo map.txt na mesma pasta do MARS
- Abra o arquivo Main.asm no MARS
- Abra o Bitmap Display e o simulador de Keyboard/Display MMIO
- Assemble e execute a Main.asm
